Opdracht 3

  1. "vertical-align: center;" is voor inline-elementen, als je tekst wil centreren moet je "text-align: center;" gebruiken.
  2. "vertical-align: middle;" is voor inline-elementen, als je tekst wilt verticaal centreren, moet je "line-height: ...px;" gebruiken.
  3. Sub-vragen:
    • De "simpele manier" om tekst verticaal te centreren is met "line-height = ...;".
    • Om meerdere regels tekst verticaal te centreren, wordt vaak de truc met "display: table" en "display: table-cell" gebruikt, waarbij "vertical-align: middle" op "table-cell-element" wordt toegepast.
  4. De regel voor de class 'tall' krijgt voorrang omdat deze specifieker is dan de algemene regel voor de class 'area'. De class 'tall' heeft een hogere specificiteit omdat het de hoogte specifiek definieert voor elementen met de class 'tall', terwijl de algemene regel voor de class 'area' een meer algemene hoogte definieert die van toepassing is op alle elementen met die class.
