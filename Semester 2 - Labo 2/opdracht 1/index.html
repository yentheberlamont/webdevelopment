Opdracht 1

1) Wat is de return value van de confirm functie als de gebruiker op een van de buttons klikt?

  • Antwoord: De returnwaarde is true als de gebruiker op ok klikt, en false als de gebruiker op cancel klikt

2.1) Wat is de return value van de prompt functie als de gebruiker een tekst intypt en op 'ok' klikt?

  • Antwoord: Als de gebruiker een tekst intypt en op ok klikt, dan is de ingevulde tekst de returnwaarde. Als de gebruiker niks invuld en toch op ok klikt, dan wordt de default waarde geretourneerd (in dit geval is dat "onbekend")

2.2) Wat is de return value van de prompt functie als de gebruiker op 'cancel' klikt?

  • Antwoord: dan wordt er "null" geretourneerd
